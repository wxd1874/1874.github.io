Flask Web

Web应用程序的本质

Web(World Wide Web)诞生最初的目的,是为了利用互联网交流工作文档

初识Flask

Flask是使用Python编写的Web微框架.Flask有两个主要依赖,一个是WSGI工具集–Werkzeug,另一个是Jinja2模板引擎.Flask只保留了Web开发的核心功能,其他的功能都又外部扩展来实现,比如数据库集成,表单认证,文件上传等.如果没有合适的扩展,你甚至可以自己动手开发.
